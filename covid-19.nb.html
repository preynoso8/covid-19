Analysis of COVID-19 time series data

STAT632

Patricia Reynoso

The Johns Hopkins University, Center for Systems Science and Engineering, has compiled a global epidemiological data set on the Novel Corona Virus (COVID-19).1 You can use the following command to load the data into R:

covid <- read.csv("https://ericwfox.github.io/data/covid19confirmed.csv")
head(covid)

The data set gives the number of confirmed cases (Value column) since 22 January 2020 for each country. As a data processing step, we need to convert the Date column from a factor type in R, to a Date object. We can use the lubridate package to do this:

library(lubridate)
covid$Date <- mdy(covid$Date)
class(covid$Date)
head(covid)

Lets start our analysis by using dplyr to subset the time series of confirmed cases in the United States:

library(dplyr)
covid_us <- covid %>% filter(Country == "US")

Question 1: Use ggplot2 to make a time series plot with the date on horizontal axis, and number of confirmed cases in the US on the vertical axis. Use geom line() when displaying the data. Next, lets investigate times series plots for other countries as well. The following command uses dplyr to subset data for Hubei province in China, Italy, South Korea, and the United States:

library(ggplot2)

covid_us%>%
  ggplot(aes(x = Date, y = Value)) + geom_line()

covid_2 <- covid %>%
